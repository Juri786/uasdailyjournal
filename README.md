Untuk menjalankan program ada beberapa yang harus dilakukan :
Mengubah file env.example menjadi .env
Memasukan api geminimu dengan cara masuk ke aistudio.google.com lalu ke bagian api dan klik generate api keys
Copy api key ke bagian GEMINI_API_KEY
Kemudian ganti bagian user ,host name, pass, dan name database. Sesuaikan dengan hosting yang dipakai dan nama databasemu
